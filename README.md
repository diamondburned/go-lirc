# lirc

[![Go Reference](https://pkg.go.dev/badge/libdb.so/lirc.svg)](https://pkg.go.dev/libdb.so/lirc)

Package lirc provides a Go client for the Linux Infrared Remote Control (LIRC)
daemon.

## Importing
